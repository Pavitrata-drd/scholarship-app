# ScholarHub — Scholar Path Finder

A scholarship discovery and application tracking platform for Indian students, built with React, TypeScript, Vite, Supabase, and shadcn-ui.

## Tech Stack

- **Vite** — fast build tool
- **React 18** + **TypeScript**
- **shadcn-ui** (Radix) + **Tailwind CSS**
- **Supabase** — auth, database, storage
- **TanStack React Query** — data fetching & caching

## Getting Started

```sh
# Install dependencies
npm install

# Start the dev server
npm run dev
```

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm run lint` | Run ESLint |
| `npm test` | Run tests |
